# 3D-туры недвижимости: подходы, сервисы, реализации

> Задача: создавать 3D-туры квартир **со смартфона, без спецоборудования**.
> Ключевой вывод: для сценария «видео с телефона → тур» правильный подход —
> **Gaussian Splatting**, проще всего брать готовым сервисом (SplatTour), а не
> строить свой GPU-пайплайн.

---

## Типы подходов

### 1. Панорамный тур (360°, переходы между точками)
- **Суть:** сферические 360°-фото в нескольких точках, связаны «переходами».
  Зритель стоит в точке, крутит головой, прыгает в соседнюю.
- **Съёмка:** 360-камера (Insta360/Theta) или телефон через приложение-фотосферу
  (Google Street View склеивает сферу).
- **Движение:** ограниченное — только между точками, без параллакса.
- **Стоимость/сложность:** низкая, дёшево, работает везде.
- **Чем реализуют:** Marzipano / Pannellum / Photo Sphere Viewer; Matterport (классика).
- **Живые примеры:**
  - Яндекс.Недвижимость: https://realty.ya.ru/d3-embedded/player/ea3817f5-fb8f-406c-b46f-ac2d676c6105
  - Демо Photo Sphere Viewer: https://photo-sphere-viewer.js.org/demos/
  - Наш прототип: `https://registrar-scheme-headsman.ngrok-free.dev/#/t/demo-apartment`

### 2. Gaussian Splatting (видео → настоящая 3D-сцена) ⭐
- **Суть:** видео-проходка → сервис склеивает множество кадров в одну связную
  3D-сцену. Свободное перемещение, параллакс, фотореализм.
- **Съёмка:** обычный телефон, медленно, 2–3 круга на разной высоте. Без железа.
- **Движение:** полное свободное.
- **Стоимость/сложность:** реконструкция на GPU (минуты). API ~$1/модель (KIRI)
  или турнкей (SplatTour от ~€4/тур).
- **Чем реализуют:** SplatTour, Real Horizons, Luma, Polycam, KIRI. Тренд 2025–2026
  (Zillow, DJI). Вьюер в браузере: `@mkkellogg/gaussian-splats-3d`, antimatter15/splat.
- **Живые примеры:**
  - https://antimatter15.com/splat/ — классический сплат в браузере
  - https://superspl.at/ — галерея + публикация/эмбед в один клик
  - https://splattour.com/en — демо-туры квартир из видео
  - https://realhorizons.ai/features/gaussian-splatting-virtual-tours/
  - https://lumalabs.ai/

### 3. Фотограмметрия / меш (3D-модель с геометрией)
- **Суть:** из фото/видео — текстурированная 3D-сетка.
- **Движение:** свободное; на плоских стенах/стекле слабее сплатов, зато есть
  настоящая геометрия (замеры, экспорт в 3D-редакторы).
- **Чем реализуют:** RealityScan (Epic), Polycam (mesh), Meshroom, KIRI (3DGS→mesh).
- **Живые примеры:**
  - https://sketchfab.com/ — тысячи фотограмметрических моделей
  - https://poly.cam/explore — галерея сканов с телефона

### 4. LiDAR / «кукольный домик» + план этажа
- **Суть:** сканирование глубины → вид сверху (dollhouse) + точный план этажа.
- **Съёмка:** нужен LiDAR — камера Matterport Pro или iPhone Pro (RoomPlan).
- **Стоимость/сложность:** выше (железо), зато точные размеры и планировка.
- **Чем реализуют:** Matterport Pro, iPhone RoomPlan, Realsee.
- **Живые примеры:**
  - https://discover.matterport.com/space/vNtptZXMm8U
  - https://gallery.we-get-around.com/examples/

### 5. Плоское видео-проходка
- **Суть:** просто записал видео обхода. Не интерактивно, не 3D.
- **Стоимость:** нулевая. Самый примитив.

### 6. Генеративный 3D (ИИ придумывает пространство)
- **Суть:** ИИ генерит проходимый мир из текста/картинки (Genie 3, Gemini text→3D).
- **Проблема:** это выдуманная квартира, а не твоя → **для объявлений не годится**;
  коммерческого API нет.
- **Пример:** https://deepmind.google/models/genie/

## Сервисы «видео → 3D-тур»

### Турнкей (загрузил видео → готовый тур со ссылкой)
| Сервис | Что делает | Цена |
|---|---|---|
| **SplatTour** ⭐ | Видео телефона → фотореалистичный 3D-тур, эмбед-ссылка, < 2 часов | Free: 40 кредитов (1 тур до 3 комнат, без карты); далее €19/мес (100 кредитов, 2–5 туров, < €4/тур) |
| **Real Horizons** | Видео/360/дрон/DSLR → гид-тур с точками, хотспотами, эмбедом, CTA, аналитикой | Про-платформа |
| **Splat Labs** | Хостинг/эмбед готового сплата: «одна строка кода → тур на сайте/Zillow/соцсетях» | Слой дистрибуции |

### API (встраивание в свой продукт)
| Сервис | Вход/выход | Цена | API |
|---|---|---|---|
| **KIRI Engine** ⭐ | видео → 3DGS (+меш: PLY/GLB/OBJ…) | ~$1/скан (1 кредит=$1, мин. 500, 10 бесплатно) | dev-API, ключ из дашборда, 7–20 мин |
| **Polycam** | видео/фото → сплаты, 15+ форматов | подписка $13–27/мес | есть |
| **Luma AI** | видео → сплаты | free ручной; API — Enterprise | ушли в генеративное видео, API мутный |

### Где Gemini полезен (умный слой, НЕ реконструкция)
Сам Gemini видео→3D не делает. Пригоден для: QC съёмки до дорогой реконструкции
(экономит кредиты), авторазметки комнат, описаний объявления, гида по съёмке.
В проекте заложен заглушками (`web/src/lib/pipeline/`), не подключён.

## Что уже реализовано в этом проекте

Прототип (mobile-first, локальный запуск + ngrok):
- **Панорамный тур** — плеер (Photo Sphere Viewer + VirtualTourPlugin, гироскоп,
  3D-стрелки), редактор сцен и переходов, шаринг. Работает.
- **3D из видео (Gaussian Splatting)** — реальный пайплайн:
  `видео → локальный бэкенд (держит KIRI-ключ) → KIRI 3DGS → скачивание .splat →
  вьюер @mkkellogg/gaussian-splats-3d`. Проверено на реальном API (submit → status →
  списание 1 кредита; баланс 10→9). Не подтверждён визуально только финальный рендер
  из-за отсутствия хорошего тестового видео.
- **Запуск:** `npm run demo` (build + Express + ngrok на :8787, один origin).

Секреты (KIRI-ключ, ngrok-токен) — в `server/.env` (gitignored).

## Рекомендация

1. **Быстро проверить качество без кода:** прогнать одно видео квартиры через
   бесплатный **SplatTour** → получить ссылку на 3D-тур.
2. **Если нужен свой продукт для арендодателей:** встраивать эмбед турнкей-сервиса
   (одна строка) ИЛИ владеть пайплайном через **KIRI API** (уже собран).
3. **Панорамы** — дешёвый запасной вариант, где не нужен «вау» и свободное хождение.
